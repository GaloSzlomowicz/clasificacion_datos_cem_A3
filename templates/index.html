Ordenador de datos CEM A3

Excel y CSV de exportación argentina → ordenados, normalizados y listos para análisis.

→ Ir a CEM A3 Mercados para descargar datos
Pasos:
  1. Descargá tu archivo desde CEM (Excel o CSV con ;)
  2. Subí el archivo acá abajo
  3. Descargá el Excel procesado listo para trabajar
Transformaciones que aplica:
  • Parsea fechas (dd-mm-aaaa) y ordena cronológicamente
  • Normaliza números argentinos (coma decimal, punto miles)
  • Quita símbolos % y convierte a float
  • Devuelve Excel listo para análisis
